# Mobile Usage

## 页面结构

- 监控：风险、传感器、摄像头、节点状态
- 控制：风险演示命令、风扇/水泵滑杆、报警开关
- 消息：MQTT 消息流

## 使用方式

1. 确保电脑端 MQTT broker 已启动
2. 手机和电脑在同一个局域网
3. 在 App 顶部输入正确的 `ws://电脑IP:9001`
4. 点击连接

## 连接成功后

你可以：

- 看实时传感器数据
- 看风险等级变化
- 看摄像头画面
- 控制风扇 / 水泵 / 报警器
- 发送演示命令
- 查看消息日志

# ESP-DL probe model artifacts

This directory is the isolated probe model location.

Current real artifacts generated by the POC export flow:

```text
model.espdl
model.info
model.json
```

Do not replace them with invented files. Regenerate them from the ONNX or
PyTorch model through ESP-PPQ with test input/output values so `model->test()`
can validate the on-board result.

Preflight entry:

```sh
firmware/tools/espdl_export_poc.sh
```
